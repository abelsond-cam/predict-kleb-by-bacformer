Use bacformer to define bacotypes to create embeddings of whole genomes, both core and accessory, and understand population structure from perspective of all genes in genomes

## Getting started

Please refer to the [documentation][],
in particular, the [API documentation][].

## Installation

You need to have Python 3.11 or newer installed on your system.
If you don't have Python installed, we recommend installing [uv][].

There are several alternative options to install Bacotype:

<!--
1) Install the latest release of `Bacotype` from [PyPI][]:

```bash

```
-->

1. Install the latest development version:

```bash
pip install git+https://github.com/abelsond-cam/Bacotype.git@main
```
